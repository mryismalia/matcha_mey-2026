# Matcha Mey — Repo Guide

## Stack
- **Laravel 12**, PHP 8.3, MariaDB 10.11
- **Filament v3** admin panel, Livewire + Blade frontend, Tailwind CSS, Vite
- **Docker** (3 containers: php-fpm, nginx, mariadb)

## Quick commands
| Alias | Expands to |
|-------|-----------|
| `dcu` | `docker compose up` |
| `dcd` | `docker compose down` |
| `dca` | `php artisan` |
| `dcm` | `php artisan make:model -msc` + Filament resource |

Container aliases (inside `php` container):
- `art` = `php artisan`
- `migrate` = `php artisan migrate`
- `fresh` = `php artisan migrate:fresh --seed`
- `t` = config:clear + pest (testdox output to `tests/report.txt`)

## Docker structure
- Source code is in `src/`, mounted at `/var/www/html` in containers
- DB port: `3306` (container) / `13306` (host)
- DB credentials: `root` / `p455w0rd`, database name = `matcha_mey`
- .env is **auto-generated by entrypoint** on container start (overwrites existing!)
- SSL certs in `nginx/ssl/` for `https://matcha_mey.test`
- Startup runs: `composer install`, `key:generate`, `migrate`, `project:init`, `storage:link`, cron

## App layout
- **Filament admin**: `app/Filament/Admin/` — panel provider at `app/Providers/Filament/AdminPanelProvider.php`
- **Livewire pages**: `app/Livewire/`
- **Blade views**: `resources/views/`
- **Routes**: `routes/web.php` (frontend), `routes/api.php`, `routes/console.php`
- **Tests**: Pest-based in `tests/`, runs with SQLite `:memory:` in CI

## Config quirks
- `CACHE_STORE=database`, `SESSION_DRIVER=database`, `QUEUE_CONNECTION=database`
- Code style: Laravel Pint preset with ordered imports (const → class → function), `one` spacing in concat, braces on next line for classes/functions
- `filament:upgrade` runs automatically on `post-autoload-dump`

## Testing
```bash
dca test              # run all tests (Pest)
dca test --filter=XXX # single test
```
Or from inside container: `t`

## Creating new resources
```bash
dcm NamaModel
# Generates: Model, migration, seeder, controller, and Filament resource
```

## Important gotchas
- **.env is overwritten every container restart** — any manual edits must go in `php/docker-entrypoint.sh` or use Docker env vars
- DB ports on host: `13306` (not standard 3306)
- Always use `dca` (or `art` inside container) instead of `php artisan` directly
- Session, cache, and queue all use database driver (run `php artisan queue:work` for queue processing)
